Project: Design and Develop an Agency Website

In 2020 I was selected as the lead for building Spotzer Agency's website and brand identity.

Our brief required an 'professional agency feel', with bold headings, lots of animation and dark colors (focusing more on blue than the orginal orange).

I built the original site Webflow, then one year later built the same site in Wordpress (using Elementor Theme). This showcase has been built using the brands style guildlines, to preserve the original design.
